# EmpathiQ

EmpathiQ is a mentoring ecosystem for teenagers, parents, and mentors. This starter scaffold focuses on:

- a role-aware data model for mobile and web surfaces
- the Teen Mission Hub experience in React Native
- the Insight Engine that translates thinking-pattern data into mentor and parent guidance

See [docs/architecture.md](/Users/mathewjose/Documents/empathiQ/docs/architecture.md) for the folder structure, database design, and system flow.
