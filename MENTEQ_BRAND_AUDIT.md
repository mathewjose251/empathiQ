## **SECTION 2: INDIAN TRADEMARK REGISTRATION**

### Official IP India Database Check:

**Platform:** [IP India Trademark Public Search](https://tmrsearch.ipindia.gov.in/tmrpublicsearch/)

**How to check:**
1. Go to IP India public search portal
2. Enter "MENTEQ" in the search field
3. Select search type: **Word Mark**
4. Click "Search"

**Expected results:**
- ✅ **AVAILABLE** = No results returned (brand is unregistered)
- ❌ **TAKEN** = Results show existing trademark registration

**If AVAILABLE:**
- File trademark application immediately
- Classes to register:
  - **Class 41:** Education; mental health support; app services
  - **Class 42:** Software development; AI services
  - **Class 44:** Mental health counseling services
- Cost: ~₹9,000–15,000 per class
- Timeline: 18-24 months to approval
- Recommend using legal firm: [LegalDev](https://legaldev.in/trademark-search), [IndiaFilings](https://www.indiafilings.com/trademark-registration), or [Marcaria](https://trademark-search.marcaria.com)

**Trademark registration link:** [Official IP India eTM Filing](https://ipindiaonline.gov.in/trademarkefiling/user/frmLoginNew.aspx)
